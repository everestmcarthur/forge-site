Frequently Asked Questions

Is Forge Hosting really free?+

Yes! Forge offers completely free game server, bot, and web hosting on shared nodes. You can deploy Minecraft, Terraria, and dozens of other servers at no cost. Premium private nodes are available from $12.99/mo for users who need dedicated resources.

What games can I host?+

Forge supports 350+ server eggs including Minecraft (Paper, Forge, Fabric, Bedrock), Terraria, Rust, ARK, Garry's Mod, CS2, Valheim, Palworld, and many more. You can also host Discord bots, websites, databases, and custom applications.

How is Forge different from Pterodactyl or Pelican?+

Forge is built on a modern panel with additional features including built-in billing, a node marketplace, ticket system, referral program, Discord integration, announcement system, and a plugin store. It's designed to be a complete hosting platform out of the box.

What is a Private Node?+

A Private Node is your own dedicated server within the Forge network. Starting at $12.99/mo, you get 10 game server slots with full control. You can add more slots in packs of 5 for $5/mo each. Save even more with annual billing at $129.90/yr.

Do I need technical knowledge?+

Not at all. Forge's panel is designed to be beginner-friendly. Create a server in 3 clicks — everything from installation to configuration is handled automatically. You get a file manager, live console, and backup system all through a web interface.
