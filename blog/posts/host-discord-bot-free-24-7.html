How to Host a Discord Bot for Free 24/7 (2026 Guide)

Heroku killed its free tier. Replit is unreliable. Running your bot on your PC means it goes offline when you sleep. Here's how to host your Discord bot for free with actual 24/7 uptime — no credit card, no gimmicks.

The Problem with "Free" Bot Hosting

If you've tried hosting a Discord bot, you've probably hit these walls:

  • Heroku — killed free dynos in 2022. Now starts at $5/mo.
  • Replit — removed always-on for free users. Bots go to sleep.
  • Railway — $5/mo credit, which your bot can easily exceed.
  • Your PC — works until you close your laptop, restart, or your internet blips.
  • Raspberry Pi — cheap but unreliable for 24/7 uptime.

Step-by-Step: Host Your Bot on Forge

1

Create a Forge account

Sign up at forged.jarviscli.dev. Free, instant, no credit card.

2

Create a new server

Click Create Server and pick your language: Node.js, Python, Java, etc. Name it after your bot.

3

Upload your bot code

Use the File Manager in the Forge panel. Drag and drop your bot files, or upload a .zip and extract it. For Git repos, use the console: git clone https://github.com/you/your-bot.git .

4

Set your bot token

Go to Startup tab and set your environment variables. For most bots, you'll set BOT_TOKEN or add a .env file via the file manager.

# .env file
DISCORD_TOKEN=your_bot_token_here
PREFIX=!
DATABASE_URL=mongodb://...
5

Install dependencies and start

In the console:

# Node.js
npm install && node index.js

# Python  
pip install -r requirements.txt && python bot.py

# Java
java -jar bot.jar

Your bot will come online in Discord within seconds. It stays running 24/7.

Pro Tips

  • Use package.json scripts — set "start": "node index.js" so Forge knows how to launch your bot
  • Crash recovery — Forge auto-restarts crashed processes. But add basic error handling to avoid restart loops
  • Git updates — pull updates via console: git pull && npm install && restart
  • Logs — the live console shows all stdout/stderr. Use it to debug issues in real-time
  • Multiple bots — free tier gives you 3 servers. Host 3 different bots, or a bot + a web dashboard + a database
