  • Always online — the server runs 24/7, even when you're not playing
  • No cost — Forge Hosting offers free Minecraft hosting with zero catches

Step-by-Step: Create Your Free Minecraft Server

1

Create your Forge account

Head to forged.jarviscli.dev and sign up. It's instant — no email verification delays. You'll land on your dashboard right away.

2

Create a new server

Click "Create Server" on your dashboard. Select Minecraft as the category. You'll see several options:

  • Paper — Best for vanilla + plugins (Spigot/Bukkit compatible). Recommended for most players.
  • Forge — For modded Minecraft (Forge mods like RLCraft, Pixelmon)
  • Fabric — Lightweight mod loader (Sodium, Lithium, etc.)
  • Bedrock — For crossplay with mobile/console players
  • Purpur — Paper fork with extra configuration options
3

Name your server and pick a version

Give your server a name (e.g., "My SMP"). Select the Minecraft version you want — the latest stable version is pre-selected, but you can choose any version back to 1.8.

4

Wait for installation (30 seconds)

Forge automatically downloads the server jar, sets up the config files, and starts your server. You'll see the console output in real-time. When you see Done! For help, type "help" — you're live.

5

Connect and play

Copy the server address shown on your server page. In Minecraft, go to Multiplayer → Add Server, paste the address, and connect. Share the address with friends so they can join too.

Optimizing Your Minecraft Server

Once your server is running, here are some quick tweaks to improve performance:

1. Adjust server.properties

In the Forge panel file manager, open server.properties and consider these changes:

view-distance=8          # Lower = less lag (default 10)
simulation-distance=6    # Reduces entity processing range
max-players=20           # Set to your actual player count
spawn-protection=0       # Remove if you want PvP at spawn
online-mode=true         # Keep true for security

2. Install essential plugins (Paper/Purpur)

  • EssentialsX — /home, /tpa, /spawn, economy
  • LuckPerms — permission management
  • WorldGuard — region protection
  • Spark — performance profiling

Upload plugin .jar files to the /plugins folder using the Forge file manager, then restart your server.

3. Pre-generate your world

Chunk generation causes the most lag on Minecraft servers. Use the Chunky plugin to pre-generate a radius of chunks:

/chunky radius 3000
/chunky start

This eliminates lag spikes when players explore new areas.

Troubleshooting Common Issues

Server won't start

Check the console for errors. The most common cause is selecting a Minecraft version that doesn't match your client. Make sure both are on the same version.

Friends can't connect

Make sure they're using the exact server address from your Forge dashboard. If they're on Bedrock trying to connect to a Java server (or vice versa), it won't work — use the matching server type.

Lag or TPS drops

Lower your view-distance to 6-8, pre-generate chunks with Chunky, and check if any plugins are causing issues with the /spark profiler command.
