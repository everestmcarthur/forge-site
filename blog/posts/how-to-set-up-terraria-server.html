How to Set Up a Terraria Server for Free (2026 Guide)

Want to explore Terraria with friends on your own server? This guide walks you through hosting a free Terraria server — including tModLoader support — in under 5 minutes.

Why Host Your Own Terraria Server?

Terraria's built-in multiplayer works via "Host & Play" — but that means the server dies when the host logs off. A dedicated server keeps the world running 24/7, lets friends join anytime, and gives you admin control over mods, world settings, and who can join.

Step-by-Step Setup with Forge Hosting

1

Create a free Forge account

Go to forged.jarviscli.dev and sign up. Takes 30 seconds — no credit card required.

2

Create a Terraria server

Click Create Server → select Terraria. For modded play, choose tModLoader instead. Name your server and hit create.

3

Configure your world

The server auto-generates a world on first boot. To customize, edit serverconfig.txt in the file manager:

worldname=MyWorld
difficulty=1        # 0=Normal, 1=Expert, 2=Master, 3=Journey
maxplayers=8
worldsize=2         # 1=Small, 2=Medium, 3=Large
password=yourpass   # Optional: protect with a password
4

Connect and play

Copy the server IP and port from your Forge dashboard. In Terraria: Multiplayer → Join via IP → paste the address. Share it with friends!

Adding Mods with tModLoader

If you chose tModLoader as your server type, you can install mods easily:

  • Open the server console
  • Use the Forge file manager to upload .tmod files to the mods folder
  • Restart the server — mods are loaded automatically
  • Make sure players have the same mods installed on their clients

Popular mod recommendations: Calamity (massive content expansion), Thorium (new classes & bosses), and Magic Storage (organization).

Optimizing Performance

  • Use Medium world size for the best balance of content and performance
  • Limit maxplayers to your actual group size
  • If running heavy mods, consider upgrading to a Private Node for dedicated resources
